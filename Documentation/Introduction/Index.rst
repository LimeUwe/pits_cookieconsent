What does it do?
================

This extension is a  GDPR_compliant_ cookie control plugin for TYPO3 websites, according to 2018 EU law.

.. _GDPR_compliant: https://eugdprcompliant.com/

PITS Cookie is a solution for every TYPO3 website owner, who plans to make their website GDPR compliant with notifying users about cookie usage and other privacy facts.
This plugin uses the most popular Cookie_Consent_ javascript to enable a super customisable cookie control widget. Whatever cookies used in the site can also be controlled thorugh this TYPO3 plugin.

.. _Cookie_Consent: https://cookieconsent.insites.com/
